"Where Vision meets Strength"

PressCoach is an AI-powered workout monitoring system that uses computer vision to track and evaluate exercise performance in real time. By utilizing a webcam and pose estimation technology, the system detects body movements, calculates joint angles, and counts repetitions to ensure proper execution of pressing exercises.

The platform integrates a web-based interface that allows users to register, log in, and remotely start workout sessions. A Python-based backend processes movement data while Firebase securely stores user and session information for progress tracking. PressCoach is designed to function both online and offline, ensuring uninterrupted performance during workouts.

With its modular architecture and real-time feedback capabilities, PressCoach provides an accessible, intelligent, and cost-efficient solution for enhancing exercise accuracy and user performance monitoring.
